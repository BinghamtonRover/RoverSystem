This was created in Visual Studio.

WHAT ISNT FINISHED
There are two things that still need to be done, The distance and angle need to be calculated using
the actual values our camera has

WHAT TO CHANGE
Run in release x64
Change VC++ Directories Include path to your opencv path
Change Library Directories Include path to your opencv path
Change C/C++ general Additional Include Directories to your opencv path
Ensure the additional Dependencies under Linker Input has your correct version of opencv

HOW TO RUN
To run, plug in as many cameras as you want and it'll automatically open as many cameras as you have available.
